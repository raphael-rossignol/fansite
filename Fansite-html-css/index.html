The Brothers War, un retour aux sources

La Guerre Fratricide (The Brothers War) revient sur cet événement fondateur de l'histoire de Dominaria : la guerre entre les frères Urza et Mishra et son impact sur le plan et le multivers.

Pour la première fois adapté en jeu, cette période de l'univers de Magic nous réserve un thème basé sur les artéfacts.

Warhammer 40k, un set brutal

La guerre sans fin du 41e Millénaire se profile à l'horizon avec la dernière collaboration Univers infinis de Magic : les decks Commander Warhammer 40,000.

Dans ce set nous laissons place à un bestiaire venant de l'univers 40k avec plus de 40 nouvelles cartes.

Ici peu de contre-sorts, place à la guerre totale.

Battle for Baldur's Gate, it's party time

Le jeu narratif Dungeons & Dragons rencontre le format le plus social de Magic dans une nouvelle expérience de crossover multijoueurs qui restera dans les mémoires.

Cette extension vous invite dans l'un des environnements les plus emblématiques de D&D, présente de nouvelles mécaniques de jeu (comme les Passés), tout ramenant des mécaniques populaires comme lancer un d20.
